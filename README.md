## Credit Card
### Supported card types
 - visa
 - mastercard
 - American Express
 - Carte Blanche
 - Diners Club
 - Discover
 - Jcb
 - Visa Master
 - Insta Payment
 - Laser
 - Maestro
 - Solo
 - Switch
 - Union Pay
 - KoreanLocal
 - BCGlobal


### Specific card type
```
iza.creditCard('Visa', 4111111111111111) // returns true
```

### Of any type
```
iza.creditCard('leroy+jenkins@super.crazy') // returns true
```
